// How it works

Three phases. Zero mercy.

01

Clone & Scan

We shallow-clone the repository and parse every Python file into an AST, detecting the framework architecture automatically.

02

Hunt Dead Code

Vulture scavenges through the AST for unused functions, classes, variables, and imports — with framework-aware whitelisting to avoid false flags.

03

Expose Zombies

We diff the declared dependencies against actual imports to surface zombie packages — installed but never used, bloating your project.
